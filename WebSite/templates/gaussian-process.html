This Section gives a brief introduction to the Gaussian process.

  • Gaussian Vector:

    • A r.v. vector \(X\) is a Gaussian Vector if \(\forall \lambda, \lambda X \sim N(\mu,\sigma^2)\)

    Theorem: r.v. vector \(X\) is Gaussian if and only if the following condition holds:

    • \(\phi_\vec{X}(\vec{u}) = \mathbb{E}(e^{i\vec{u}\times\vec{X}})=exp(i\vec{u}\times\vec{\mu}-\frac{1}{2}\vec{u}^TC\vec{u})\)

      • Here \(\mu \in \mathbb{R}^n\) and \(C\) is positive semi-definite of size \(n \times n\)

    • \(\vec{X} = A\vec{x}^0 + \vec{\mu}\)
      • Where A can be any matrix of \(n \times n\) and \(\vec{x}^0\) is a standard normal vector
    • Let \(X1, X2\) be r.v. from normal distribution and \(corr(X1, X2) = 0\) then X1, X2 are independent if and only if \((X1, X2)\) is a Gaussian vector

  • Gaussian Process:

    • A Gaussian process \(X_t\) is a stochastic process where \(\forall t1, t2...tn, (X_{t1},X_{t2}...X_{tn})\) is a Gaussian Vector

    Theorem: Let \(m:R_+ \rightarrow R\) and \(K: R_+\times R_+ \rightarrow R\) then there exists an Gaussian process \(X_t\) such that \(\mathbb{E}X_t = m(t)\) and \(Cov(X_t, X_s) = K(t,s)\)

  • Brownian Motion (Wiener Process):

    • Definition 1: Brownian motion is a Gaussian process s.t. \(m(t) = 0, K(t,s) = min(t, s)\)

    Definition 2: Brownian motion is a stochastic process s.t.

    • \(B_0 = 0\)
    • \(B_t\) is independent
    • \(B_t-B_s \sim N(0,t-s)\)

    Kolomogorov Continuity Theorem:

    \(X_t, Y_t\) are stochastic equivalent if \(P(X_t = y_t) = 1, \forall t \geq 0\)

    If: $$\exists C, \alpha, \beta > 0 s.t. \mathbb{E}|X_t - X_s|^\alpha \leq C|t -s|^{1 + \beta} ,\forall t,s \in [a,b]$$ Then \(\exists y_t\) stochastic equivalent to \(X_t\) s.t. \(y_t\) has continuous trajectory. We call \(y_t\) the continuous modification. Brownian motion has continuous modification

  • Brownian Motion Properties:
    • Quadratic Variation:
      • Given:$$\Sigma_{k=1}^nt_k = t$$ We have: $$lim_{n->\infty}\Sigma_{k=1}^n(B_{t_k} - B_{t_{k-1}})^2 = t$$
    • Brownian motion is everywhere continuous but no where differentiable.
    • \(lim_{t->\infty}\frac{B_t}{t} = 0\)
