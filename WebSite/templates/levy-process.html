Here is a brief overview of the Levy process. Let's start with some comparison

Poisson ProcessBrownian MotionLevy Process
\(N_0=0\)\(W_0=0\)\(L_0=0\)
Has Independent Increment Has Independent Increment Has Independent Increment
Has Stationary Increment Has Stationary Increment Has Stationary Increment
\(N_t - N_s \sim Pois(\lambda(t-s))\) \(W_t - W_s \sim N(0, t-s)\) \(L_t - L_s \sim P(t-s)\)
Where \(P\) is called infintely divisable distribution

This gives us a defition of Levy Process which is 0 at time 0, has Independent and Stationary increment, has the above infintely divisable distribution and stochastic continuous. The trajectory of a Levy process is called Cadlag function. \(f\) is a cadlag function if:

$$\exists lim_{s \rightarrow t-}f(s)=f(t-), \exists lim_{s \rightarrow t+}f(s) = f(t+) = f(t-)$$
  • Infintely Divisable Distribution

    • \(\xi\) has infintely divisable distribution if: \(\forall n \geq 2, \xi \xrightarrow{d} \Sigma Y_i\) where \(Y_1, Y_2...\) are i.i.d and:

    $$\phi_{\xi}(u) = (\phi_{Y_i}(u))^n$$

    A stable distribution are \(\xi_1, \xi_2...\) are i.i.d to \(\xi\)

    $$\Sigma \xi \xrightarrow{d} a_n\xi + b_n$$

    A stable distribution means the sum of the rv follow the same distribution as the individual distribution. Some example distribution like: Normal, Cauchy, Gamma, Exponential, Negative Binomial, Geometric, Poisson, Compound Poisson distribution are infintely divisable distribution but among them only Normal and Cauchy are stable distribution. Bernoulli and Uniform distribution are not infintely divisable

    Property of infintely divisable distribution:

    • \(\phi(u) = 0\) does not have any real solution
    • \(supp(\xi)\) is unbounded. Where \(supp(\xi)\) is all set contain \(\xi\) such that the measure of each set are strongly positive.

  • Characteristic Exponent of Levy Process

    • Definition: For any process \(L_t\):

    $$\phi(u) = \mathbb{E}[e^{iuL_t}] = e^{t\psi(u)}$$

    Here \(\psi(u)\) is the Characteristic exponent. For a Levy Process, the distribution of \(L_t\) is determined by \(L_1\) where \(\mathbb{E}L_t = t \mathbb{E}L_1, Var(L_t) = tVar(L_1), K(t,s) = min(t,s)VarL_1\)

  • Levy Measure:

    • Levy measure is the expectation of the amount of jumps where the size of jump belongs to the Borel non zero set between \(t \in [0,1]\). For Brownian motion, the Levy measure \(\nu\) is 0. For Poisson process \(\nu(B) = 0\) if \(1 \not\in B\) and \(\nu(B) = EN_1=\lambda\) if \(1 \in B\) For compound Poisson process: \(\Sigma_{k=1}^{N_t}\xi_k\) we have: \(\nu(B) = \int_B\lambda P_{\xi}(x)dx\) where \(\lambda P_{\xi}\) is called Levy density

  • Levy Khintchine Theory:

    • The Characteristic Exponent of any Levy process can be fomulated as:

    $$\psi(u) = iu\mu - \frac{1}{2}u^2\sigma^2 + \int_R(e^{iux}-1-iux\mathbb{1}[|x| < 1])\nu(dx)$$ Where \(\mu \in R, \sigma > 0, \nu\) is a Levy Measure and we call \((\mu, \sigma, \nu)\) a Levy-Khintchine triplet.

    The above tells us a Levy Process \(X_t = \mu t + \sigma W_t + J_t\) where \(W_t\) is a Brownian motion and \(J_t\) is a jump process.

    The Levy process is a bounded variation if and only if \(\sigma = 0\) and \(\int_{|x| < 1}\nu(dx) < \infty\) in this case:

    $$\psi(u) = iu\tilde{\mu} + \int(e^{iux} - 1)\nu(dx)$$
  • Levy Based Models:
    • Time Changed Model:

      • Let \(log\frac{S_t}{S_0} = W_T(s)\) and \(T(s)\) be the cumulative amount of transactions from 0 to s

      Monroe's Theorem: The class of all brownian motion coicide with all semi martingales.

      $$\phi_{X_t(s)}(u) = L_{T(s)}(-\psi(u))$$
