This section gives a brief over view of the Poisson Process

  • Definition
    The Poisson process is a renewal process such that \(\xi_i\) has exponential distribution. (i.e. \(\xi_i \sim p(x) = \lambda e^{-\lambda x} x > 0\)). Here \(\lambda\) means intensity rate.
  • Theorem I: distribution function of \(S_n\) $$S_n \sim 1 - e^{-\lambda x}\Sigma_{k = 0}^{n - 1}\frac{(\lambda x)^k}{k!}$$ $$P_n(x) = \lambda e^{-\lambda x}\frac{(\lambda x)^{n-1}}{(n-1)!}$$
  • Theorem II: distribution function of \(N_t\) $$P(N_t = n) = e^{-\lambda t}\frac{(\lambda t)^n}{n!}$$

    That is \(N_t \sim Poisson(\lambda t)\)

  • Memoryless Property
    • A random variable X possesses the Memoryless property if: $$P[X > u + v] = P[X > u]P[X > v]$$ If \(P[X > v] > 0\) Then: $$P[X > u + v | X > v] = P[X > u]$$
  • Theorem III
    Let X be a random variable with density \(p(x)\) then
    X is memoryless \(\iff p(x) = \lambda e^{-\lambda x}\)
  • Limit of Poisson Process: $$lim_{h\rightarrow 0}P(N_{t+h} - N_t = 0) = 1 - \lambda h + O(h)$$ $$lim_{h\rightarrow 0}P(N_{t+h} - N_t = 1) = \lambda h + O(h)$$ $$lim_{h\rightarrow 0}P(N_{t+h} - N_t \geq 2) = O(h)$$
  • Other Definition

    • We can also define poisson process as \(N_0 = 0\) and \(N_t\) has independent and stationary increment and :

    • \(N_t - N_s \sim Poisson(\lambda(t - s))\)
    • or \(lim_{h\rightarrow 0}\frac{P(N_{t+h} - N_t \geq 2)}{P(N_{t+h} - N_t = 1)} = 0\)

  • Non-Homogenous Poisson Process

    Definition: Let \(\Lambda(t)\) be a differentiable increasing function and \(\Lambda(0) = 0\). Then \(X_t = N_t\) is a Non-Homogenous Poisson Process:

    • \(N_0 = 0\)
    • \(N_t\) has independent increment
    • \(N_t - N_s \sim Poisson(\Lambda(t) - \Lambda(s))\)

    Here \(\lambda(t) = \Lambda'(t)\) is called intensity function
    Propeties of the process:
    • \(\mathbb{E}[N_t] = \Lambda(t)\)
    • \(N_{\Lambda^{-1}(t)}\) is a Homogenous Poisson process
